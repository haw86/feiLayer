基础参数

type 层类型 title 标题 content 内容 shade 是否显示遮罩 shadeClose 是否遮罩关闭层 skin 自定义风格 btn 按钮 time 自动关闭的时间

回调方法

success 弹出成功后触发 yes 确定 no 取消 end 关闭后触发

介绍

feiLayer mobile是为移动设备（手机、平板等webkit内核浏览器/webview）量身定做的弹层UI。不支持IE之类的浏览器内核。另外，该组件采用原生JavaScript编写，无需依赖任何第三方库。

快速上手

1.下载好文件后，部署到您的项目里面，然后引入下面两个文件就可以了： 

./feiLayer/default.css
./feiLayer/feiLayer.min.js
两个引入时，注意文件的路径不要弄错了！

2.调用如下代码，就可以直接使用了，非常简单！您可以通过提供的Api调整到需要的功能： 

feiLayer.open({
    content:"晚上好，春天小姐！"
});
试一下

type层类型

类型：Number，默认：1
提供3中层类型，1（信息框）、2（提示框）3（页面层）
如下： 

feiLayer.open({
    type:2,
    content:"来自“春天小姐”的第一个提示框"
});
试一下

title标题

类型：String，默认：不显示
如需要显示传值即可
默认无标题： 

feiLayer.open({
    content:"无标题！",
    btn:["朕知道了"]
});
试一下

自定义标题： 

feiLayer.open({
    title:"温馨提示",
    content:"人家是标题，好吧",
    btn:["朕知道了"]
});
试一下

content内容

类型：String，默认：""
如需要显示传值即可:

feiLayer.open({
    content:"我是被带进来的内容！",
    btn:["朕知道了"]
});
试一下

shade是否显示遮罩

类型：Bool，默认：true
如果不需要遮罩，设置 false 即可:

feiLayer.open({
    shade:false,
    content:"无遮罩！",
    btn:["朕知道了"]
});
试一下

shadeClose是否使用遮罩触发关闭层

类型：Bool，默认：true
如果不需要该功能，设置 false 即可:

feiLayer.open({
    shadeClose:false,
    content:"遮罩不能关闭！",
    btn:["朕知道了"]
});
试一下

skin自定义calss

类型：Bool，默认：true
如果不需要该功能，设置 false 即可:

feiLayer.open({
    skin:"layout-class",
    content:"自定义风格Class！",
    btn:["朕知道了"]
});
试一下

btn按钮

类型：Array，默认：["取消","确定"]
如果只需要1个自定义按钮，设置 ["确定"]即可:

feiLayer.open({
    content:"一个按钮！",
    btn:["确定"],
    yes:function(){
        feiLayer.open({
            type:2,
            content:"yes 回调"
        });
    }
});
试一下

如果只需要2个按钮，设置 ["取消","确定"]即可:

feiLayer.open({
    content:"2个按钮！",
    btn:["确定"],
    yes:function(){
        feiLayer.open({
            type:2,
            content:"yes 回调"
        });
    },
    no:function(){
        feiLayer.open({
            type:2,
            content:"no 回调"
        });
    }
});
试一下

time自动关闭的时间

类型：Number，默认：无
如果需要该功能，设置 time 即可:

feiLayer.open({
    time:3000,
    content:"3秒后自动关闭"
});
试一下
